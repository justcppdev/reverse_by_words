# reverse_by_words

Необходимо развернуть строку по словам.

### Входные данные
Во входных данных задана строка, содержащая числа разделенные пробелами

### Выходные данные
Строка, полученная путем разворота по словам.

### Примеры
#### входные данные
`123 45 67 8 910`
#### выходные данные
`910 8 67 45 123`
#### входные данные
`1  23 4   56`
#### выходные данные
`56   4 23  1`
